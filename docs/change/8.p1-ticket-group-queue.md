# P1 Ticket Group And Queue

更新时间：2026-04-20

## 本次实现范围

本次只实现 P1 的基础配置能力：

- 工单组 `ticket_group`
- 工单队列 `ticket_queue`
- 工单主表队列绑定字段 `ticket.group_id`、`ticket.queue_id`
- 工单队列绑定接口

不实现：

- 自动分派。
- SLA 计算。
- Agent 自动操作队列配置。

## 模块落位

### domain

- `TicketGroup`
- `TicketQueue`
- `TicketGroupMapper`
- `TicketQueueMapper`
- `TicketGroupMapper.xml`
- `TicketQueueMapper.xml`

### biz

- `TicketGroupService`
- `TicketQueueService`
- `TicketGroupRepository`
- `TicketQueueRepository`

业务规则：

- 工单组编码创建后不允许修改。
- 队列编码创建后不允许修改。
- 队列必须归属于存在且启用的工单组。
- 写操作必须由 ADMIN 执行。

### api

- `TicketGroupController`
- `TicketQueueController`
- `TicketController.bindTicketQueue`
- `TicketGroupRequest`
- `TicketQueueRequest`
- `BindTicketQueueRequestDTO`
- `UpdateEnabledRequest`
- `TicketGroupVO`
- `TicketQueueVO`

## 接口

### 工单组

- `POST /api/ticket-groups`
- `PUT /api/ticket-groups/{groupId}`
- `PATCH /api/ticket-groups/{groupId}/enabled`
- `GET /api/ticket-groups/{groupId}`
- `GET /api/ticket-groups`

### 工单队列

- `POST /api/ticket-queues`
- `PUT /api/ticket-queues/{queueId}`
- `PATCH /api/ticket-queues/{queueId}/enabled`
- `GET /api/ticket-queues/{queueId}`
- `GET /api/ticket-queues`
- `PUT /api/tickets/{ticketId}/queue`
